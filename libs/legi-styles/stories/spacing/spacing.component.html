Spacing

The spacing system uses the one from Bootstrap, customized with our own scale. Most spacing classes are available as placeholders as well, like so: 

@import 'components-pack';

.my-div {{ '{' }}
  @extend %mb-3;
}
.p-{{ spacing.name }} .m-{{ spacing.name }} {{ spacing.multiplier }}rem ~{{ spacing.multiplier * 16 }}px
